# OpenBack

OpenBack is a fast-paced territorial real-time strategy game about expansion, economy, diplomacy, and combined-arms warfare. Players compete across detailed world maps, grow organic borders, construct infrastructure, form alliances, and deploy land, sea, air, and nuclear forces in solo, private multiplayer, and ranked matches.

The game features deterministic simulation, bots and nations, shareable multiplayer lobbies, matchmaking, clans, structures, railroads, ships, aircraft, tanks, missile defense, replays, and configurable match modifiers.

OpenBack builds on the open-source OpenFrontIO codebase while providing its own branding and gameplay additions.
