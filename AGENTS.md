# OpenBack project instructions

- Every user-facing change must add or update an entry at the top of `resources/changelog.md` as part of the same commit.
- Release notes must use an OpenBack version, describe what players actually receive, and credit **frootz jhklphy**.
- Keep release notes readable in the existing in-game Release Notes modal; do not introduce a separate visual style.
- Preserve the AGPL, corresponding-source availability, required copyright notice, asset-license notices, and contributor attribution.
- Do not add optional OpenFront Discord, Reddit, Wiki, store, or promotional links.
- Do not publish setup, local-run, self-hosting, or deployment instructions unless the user explicitly asks for them.
