OpenBack privacy notice

This repository is a local, renamed open-source game build. It is not the OpenFront hosted service and is not operated by OpenFront Inc.

Local build

OpenBack does not include the upstream advertising or analytics boot scripts. Solo game state is processed in your browser. Local lobbies use the server you start from this source tree.

Browser storage and network features

The client may store settings, a generated local player identifier, and other preferences in browser storage. Optional links and account, leaderboard, wiki, tutorial, or source-code features can connect to third-party or upstream services when you choose to use them; those services have their own policies.

Hosting OpenBack

If someone deploys OpenBack publicly, that operator is responsible for publishing an accurate privacy policy for the data and services used by that deployment.
