# Open Map One

- **Author:** Darklighter Designs
- **Published:** 2017
- **Source:** https://opengameart.org/content/open-map-one
- **License:** Creative Commons Attribution 3.0
- **License text:** https://creativecommons.org/licenses/by/3.0/

The bundled `open-map-one.png` is used as the coastline and land/water source
for OpenBack's modified Shattered Expanse terrain. OpenBack changes its scale,
visual terrain treatment, spawn layout, nations, and gameplay data.
